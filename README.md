## 🌟 Overview
This DApp allows creators to secure their intellectual property by hashing files locally and registering the unique digital fingerprint on the blockchain. It features a modern Glassmorphism UI and seamless MetaMask integration.

## 🛠 Tech Stack
- **Smart Contract:** Solidity (^0.8.1)
- **Frontend:** Vue 3 (Composition API), Vite
- **Web3 Library:** Ethers.js (v5)
- **Styling:** CSS3 with Glassmorphism effects
- **Network:** Polygon Amoy Testnet

## 📝 Smart Contract Details
- **Contract Address:** `0xYOUR-CONTRACT-ADDRESS-HERE`
- **Network:** Polygon Amoy
- **Features:** 
  - Proof of Existence (PoE) via SHA-256
  - Automated payment (0.01 POL fee)
  - Duplicate registration prevention
